# @vuepress/tsconfig

> TSConfig for VuePress Team

**DO NOT use it in your own project if you don't know what it's for**

## Install

```sh
npm i -D @vuepress/tsconfig
```

## Usage

In `tsconfig.json`:

```json
{
  "extends": "@vuepress/tsconfig/base.json"
}
```

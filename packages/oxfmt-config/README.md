# @vuepress/oxfmt-config

> Oxfmt config for VuePress Team

**DO NOT use it in your own project if you don't know what it's for**

## Install

```sh
npm i -D @vuepress/oxfmt-config
```

## Usage

```ts
import { defineConfig } from 'oxfmt'
import { vuepress } from '@vuepress/oxfmt-config'

export default defineConfig(vuepress())
```

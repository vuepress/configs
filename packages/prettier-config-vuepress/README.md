# prettier-config-vuepress

> Prettier config for VuePress Team

**DO NOT use it in your own project if you don't know what it's for**

## Install

```sh
npm i -D prettier-config-vuepress
```

## Usage

In `package.json`:

```json
{
  "prettier": "prettier-config-vuepress"
}
```

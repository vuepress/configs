# @vuepress/prettier-config

> Prettier config for VuePress Team

**DO NOT use it in your own project if you don't know what it's for**

## Install

```sh
npm i -D @vuepress/prettier-config
```

## Usage

In `package.json`:

```json
{
  "prettier": "@vuepress/prettier-config"
}
```
